# Code Organization

Clone the repository into the `src` folder of your catkin workspace. Like this
```
cakin_ws/
	build/
		...
	devel/
		...
	src/
		CARDSflow/
			...
		gelatatio-boy/
			<subpackages of our project>
		<further packages if required>
```

Catkin build the packages recursively.
